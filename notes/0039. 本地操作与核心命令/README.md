## 2. 🫧 评价

这一章提到的一些核心命令有些并不常用，本章介绍的命令中，个人日常使用频率最高的是下面这 3 条：

```bash
git add . # 添加所有变更文件到暂存区
git commit -m"update" # 提交到本地仓库
git push # 推送到远程仓库
```
