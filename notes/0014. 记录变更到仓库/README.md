## 1. 🎯 本节内容

- 检查文件状态（`git status`）
- 跟踪新文件与暂存修改（`git add`）
- 忽略文件（`.gitignore` 配置）
- 查看差异（`git diff` 与 `git diff --staged`）
- 提交更新（`git commit`）
- 跳过暂存区直接提交（`git commit -a`）
- 移除文件（`git rm`）
- 移动或重命名文件（`git mv`）
